PRÁCTICA: Desarrollo de un sitio web con Bootstrap, jQuery y Buenas Prácticas
El objetivo de esta práctica es que los alumnos desarrollen un sitio web aplicando los
conceptos de HTML5 junto con el uso de frameworks y librerías como Bootstrap y jQuery. Se
espera lograr un diseño responsivo, interactivo y moderno optimizando el desarrollo. Adaptar
el trabajo de HTML-CSS.
El sitio debe adaptarse correctamente a: Mobile , Tablet , Desktop , Pantallas grandes.
Estructura del Sitio: Turismo
1. Home
• Navbar responsive (Bootstrap. Dropdown / Mega menú)
• Hero con imagen/video + texto animado (jQuery)
• Cards de destinos con hover
• Contador animado (jQuery)
• Carousel de testimonios
• Footer:
• Formulario
• Redes sociales
• Mapa embebido
2. Destinos
• Filtros dinámicos con jQuery (.filter(), .hide(), .show())
• Grid responsive ( Adaptación a 1, 2 o 3 columnas según pantalla)
• Cards con efectos de zoom usando CSS + jQuery
• Tabla de precios responsive (table-responsive)
3. Agencias
• Cards con efecto flip (CSS + jQuery toggle)
• Sistema de rating con estrellas (jQuery)
4. Contacto
• Formulario con validación en tiempo real (.on('input'), .val())
• Spinner de carga (spinner-border)
• Modal de confirmación
• Diseño responsive (Flexbox + Grid de Bootstrap)
5. Precios
• Tabla comparativa
• Hover dinámico
• Tooltips (Bootstrap + jQuery)
6. Blog
• Layout tipo revista
• Filtro por categorías (jQuery)
• Comentarios simulados
• Animaciones al hacer scroll
7. Módulo Educativo: Phishing
NO se permite robo de datos reales.
